ClockEasy

Full-stack Developer

Research Website

VesselVerse

Project website for a collaborative framework and dataset for brain vessel annotation

I-VESSEG · EURECOM

UX/UI and Front-end Development

Designed and developed the website for VesselVerse, a research project centered on collaborative vessel annotation. The site communicates the dataset, framework, and research context behind a platform built to support multi-expert annotations, consensus generation, and versioned updates for neurovascular imaging data.

Project Focus

The project had to present a medical imaging initiative in a way that remained clear to research collaborators while also making the platform and dataset immediately legible from the website itself.

Stack

  • Static front-end architecture
  • Responsive UI design
  • Research communication and content structuring
  • Project and dataset presentation

Highlights

  • Presented the framework as both a dataset and a collaborative annotation workflow
  • Structured the site around research goals, platform access, and publication context
  • Connected the product-facing website to the broader I-VESSEG research initiative
